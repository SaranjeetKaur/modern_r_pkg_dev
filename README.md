
# Modern R Package Development: Scrollytelling version

This repository provides a scrollytelling version of "Modern R Package Development" inspired and based on [Athanasia Monika Mowinckel](https://drmowinckels.io/)'s [Advent Calendar 2025 series](https://www.linkedin.com/posts/drmowinckels_rpackageadvent2025-rstats-rpackageadvent2025-activity-7400837138604290048-dOt4). The content has been adapted to create an interactive experience that guides users through the principles and practices of modern R package development. It is best viewed on laptop or desktop devices for optimal experience.
